# Student Grade System

This is a simple Java project that stores students and their grades.

## Description

The program creates student objects, calculates their average grade, and checks if each student has passed or failed.

A student has passed if the average grade is 50 or higher.

## Files

- `Main.java` - Runs the program
- `Student.java` - Contains the Student class

## Features

- Stores student name
- Stores student ID
- Stores multiple grades
- Calculates average grade
- Checks pass/fail result
- Prints student information

## How to run

1. Open the project in IntelliJ IDEA.
2. Open `Main.java`.
3. Press the green run button.
